Jamal's Conversion App 1.0 Readme

Welcome to Jamal's Conversion App 1.0. Here's how to use the app:


First you will need something to convert. Simply enter the number of units you want to convert then use the dropdown menu to choose what type of units you are using to the type of units desired then everything will be converted automatically. That's it! Also remember these rules:



1) Volume is how much something holds.

2) Length is how long or short something is.

3) Mass is how much something weighs.
